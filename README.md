# Beauty Brand Purchase-Legitimacy Lexicon（BPLL）
## **美妆品牌购买合法性词典 —— 危机下品牌资产韧性的话语测量**
### ver 1.0

---

## 1. 项目概述

本项目以 2023 年日本福岛核污染水排海事件为案例，基于微博与小红书的公开消费者发言，构建一套可复用的中文话语测量工具，期待解决一个问题：当品牌遭遇危机时，哪些顾客流失、哪些顾客留存、留存者以何种理由为品牌辩护。

项目交付两项成果：

1. **购买合法性词典**：一套开源的中文词表与配套分类规则，用于识别消费者发言中的购买理由类型与危机反应类型。词典的自动分类能力以人工标注基准为上限：每个类别的判断准确率经独立验证并公开报告，未达标准的类别不参与自动化统计。
2. **危机韧性诊断报告**：基于五个品牌（资生堂、SK-II、花王；珀莱雅、蜂花作为国货对照组）在事件前后共三个时段的话语结构变化，检验四项预先登记的研究假设。

## 2. 业务背景：原产国风险与品牌基本盘

核污染水排海并非孤立事件，而是一类反复出现的风险的一次显现。每当中日历史议题升温、日本政坛官员言论等事件发生时，针对日本的民族主义情绪极有可能带来对日系品牌的抵制情绪。此类风险的共同特征是：与品牌自身经营行为无关，仅与品牌的原产国身份绑定。品牌无法通过改进产品来消除这一风险敞口，即所谓原产国风险（country-of-origin risk）。

面对无法根除、只能反复承受的风险，品牌方的理性策略是准确认识自身的顾客基本盘。具体而言，需要回答三个问题：

1. **流失的成因结构**：顾客离开是因为对产品安全的担忧，还是因为消费行为本身被赋予了立场含义？这两种流失的可挽回性与应对方式截然不同。
2. **留存的理由结构**：危机之后仍为品牌辩护的顾客，其辩护逻辑是什么？留存者的辩护理由构成了品牌穿越危机周期的根基。
3. **沟通策略的适配性**：当危机发生时，何种危机沟通话术能够保护基本盘？本项目的假设四（辩护有效性）直接针对此问题。

上述框架适用于任何承受原产国风险的品牌，本项目的方法设计以可复用为目标。

## 3. 分析框架：购买合法性的两种类型

消费者为高价美妆品牌支付溢价的理由，可归纳为两种类型：

**价值认同型（value-based legitimacy）**：购买理由指向产品本身的属性，例如成分、安全性、功效、科研背书。典型表达：“这款产品成分温和，烂脸期全靠它”“是真的有实验室有专利的”。

**社会信号型（signal-based legitimacy）**：购买理由指向产品的社会含义，使用该品牌向他人传递的身份信息。典型表达：“这个牌子是贵妇标配，拿得出手”“其实也没有很好用，但毕竟是大牌，拿出去有面子”。

在此需事先声明三点方法论：

- 本项目的研究与分类对象是**发言**而非**用户**。笔者认为，同一用户可以在不同发言中使用不同类型的理由，因此本项目不对任何个体进行归类或追踪。
- 该二分类以研究假设的形式提出，并已在语料精读阶段获得初步的效度支持：消费者话语中存在明确区分两种逻辑的元表述（如“我们是抵制，不是害怕”），表明该区分是消费者的本土范畴而非研究者的外部强加。
- 类别的完整定义、边界判断规则与真实例句见编码手册（docs/codebook_v1.md）。

## 4. 研究缘起：化妆品与药品的反差

值得一提的是，核污染水排海后，社交平台出现大量“避雷日本化妆品”内容，但未见成规模的“抵制日本药品”的讨论。这不由得引发了笔者的思考：若消费者的规避行为完全由安全担忧驱动，摄入类产品理应引发比外用类产品更强的规避，然而笔者观察到的现实却截然相反。

财务数据为这一反差提供了参考性佐证：2023 年第三季度，资生堂中国区销售额同比下滑约 9%，全年核心营业利润预期下调 42%，双十一预售首日 GMV 同比下滑 74.1%；同年，以健康用品为主业的小林制药中国销售收入同比增长 9.4%，并保持连续 26 年净利润增长。

需要声明的是，该对比的作用限于提出研究方向。对药品豁免现象存在竞争性解释，例如药品缺乏替代品、属于必需品、消费不可见，而本项目无药品语料，不对这些解释做出裁决，并且笔者认为该财务对比的证明力也存在明确边界：其一，日本药品与保健品的在华消费大量经由代购渠道完成，相应收入计入日本本土销售额，各药企的中国区财报系统性低估中国消费者的实际购买量，且低估幅度不可知；其二，以处方药为主业的企业（如第一三共）经由医院采购渠道销售，其在华收入不受消费者情绪影响，不构成有效对照；其三，小林制药为单一公司、年度颗粒度，且其在华主力产品（暖宝宝、退热贴等）严格而言属健康用品。因此本项目不将以上财报数据作为该论点的核心证据，反差的主要证据仍是话语层面的：公开社交平台上不存在与美妆抵制规模相当的药品抵制讨论。

笔者对这一反差的解释假说是：**化妆品带有社交品属性，药品没有**。化妆品的使用是可见的、可被他人评价的，产品除使用价值外还承载身份含义；药品的使用是私密的，接近纯粹的生活用品。在此情况下，身份含义便有了双刃剑属性：常态下，“使用日系高端品牌”是正向信号，但当民族情绪升温后，同一信号便立刻翻转为政治立场质疑的对象。

## 5. 研究假设

四项假设在数据分析开始前登记，每项均附有可观测的检验预期。

**假设一（污名机制）**：民族情绪主要冲击社会信号型合法性。民族情绪攻击的对象不是产品，而是“使用日货”这一行为的社会形象；以社会信号为购买理由的话语，在形象翻转后失去支撑。<br>
 * 检验预期：事件爆发期（T1），信号类话语转向负面，出现将消费选择与立场绑定的攻击性表达。

**假设二（恐慌机制）**：安全担忧主要冲击价值认同型合法性。以成分安全与功效为购买理由的话语，直接暴露于核污染传闻的打击面。<br>
 * 检验预期：T1 期价值类话语中出现对原料安全的系统性担忧。

<br>

**假设一与假设二预期同时成立，检验重点为两种机制的相对强度。**

<br>

**假设三（构成假设）**：品牌承受的冲击结构由其顾客构成决定。事件前（T0）话语中信号类占比越高的品牌，受污名机制的冲击越重；价值类占比越高的品牌，其讨论区出现理性辩护的比例越高。<br>
 * 操作化：以 T0 期各品牌两类话语的占比作为顾客构成的代理变量，检验其对 T1 期冲击结构的预测力。<br>
 * 事前登记的具体预测：花王以日用品为主业，信号属性为日系三品牌中最弱，其所受攻击中立场类占比应为三者最低。<br>
 * 辅助验证：检索各品牌公开披露的顾客调研数据，与 T0 测量交叉比对；两者冲突时以 T0 语料测量为准，冲突情况如实报告。<br>
 * 样本限制：品牌数为五，本假设仅做方向性检验，不做统计推断。

**假设四（辩护有效性）**：危机沟通话术的有效性取决于其回应的质疑类型。产地转移、批次检测、国际标准认证等辩护话术回应的是安全质疑，预期仅在价值类讨论语境中获得平和回应；面对立场类质疑时，同类话术预期无效。
 * 检验路径：利用数据中的回复链字段（parent_post_id），比较辩护类发言在两类语境下的分布与所获回应。

## 7. 方法与质量控制

测量方法采用话语类别占比而非通用情绪分数，原因是本语料中文本情绪与品牌态度频繁背离（对品牌负面消息的正面喝彩、对产品的焦虑式赞美，例如“干得漂亮”指的是品牌销量下滑，感情是喝彩式的，但对品牌形象却是负面的），且评论文本中位长度约 13 字，通用情绪工具误判风险高。<br>
类别占比的每一步（词典构建、消歧规则、人工标注验证、置信区间）均有明文规则，详见 METHODS.md。

质量控制要点：

- **词典准入与审计**：种子词经语料精读产生、按明文准入标准筛选，并经出现频次审计；语料实证不足的词条在词表中公开标记（prior_driven），保留供跨语料复用，不参与本研究的证据链。
- **人工标注验证**：500–800 条样本将由三名标注者独立标注，报告标注者间一致性；词典逐类别与人工标注对照，查准率低于 0.75 的类别不进入自动化统计。
- **结果密封**：涉及时段对比的分析在研究设计定稿与词典验证完成前不运行；分析脚本的开发与测试仅使用安慰剂数据（品牌标签重排或单时段切片），全部运行留有日志。

## 9. 研究局限

1. **发声者偏差**：语料经关键词检索获得，仅覆盖公开发言者，因此沉默的留存者与沉默的流失者均不可见，所有结论限定于公开发言人群。覆盖沉默人群需引入问卷与消费面板数据，属于本方法的后续扩展方向。
2. **发言与人的区分**：本项目主要测量话语构成的变化，不支持任何个体层面的流失或留存推断。
3. **品牌样本量**：假设三的品牌层面检验仅有五个观测单位，结论为方向性而非统计性论断。
4. **事件混杂**：国货对照组（蜂花）的爆发期话语与同期发生的另一营销事件存在重叠，相关条目已标记并在分析中分列处理。
