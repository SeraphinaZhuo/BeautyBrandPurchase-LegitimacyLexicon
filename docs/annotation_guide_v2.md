# 标注指南

**BPLL 项目 | 版本 v2.0**

本指南写给参与本项目文本标注的标注者。相比上一版，本版为每个容易混淆的标签补充了【算 / 不算】的对照示例，并把原来的无法判断拆分为两个更明确的栏目。阅读本指南后，你应当能够独立完成标注工作。

---

## 1. 你的任务是什么

你会收到一份 Excel 表格，里面有若干条来自社交媒体的真实发言，内容都与美妆、日化品牌有关，采集自二〇二三年下半年。

你的任务是：**逐条阅读发言，判断每条发言里出现了哪些类型的表达，在对应的列里选择✓。**

几点总原则：

* **只根据发言写出来的内容判断**，不要猜测发言者没有说出口的想法。发言里没有明说的，即使你觉得能推出来，也不要标。
* **一条发言可以同时属于多个类型**，符合几个就标几个；也可以一个类型都不属于，全部留空即可。
* **只要发言明确指向了某个类型，无论语气强弱，都要标。** 不要因为“感觉还没到那个程度”就不标。强弱的差别由后续分析处理，标注者只需要根据发言判断属于某种类型即可。
* **初步判断不了的，请用新增的两个栏目**（主题无关 / 上下文缺失），不必硬塞进某个态度标签。只有当一条发言既不属于任何态度标签、又不属于主题无关或上下文缺失时，才勾“无法判断”，并在备注写明原因。（预期这种情况很少）
* **请独立完成**。在标注全部结束之前，请不要与研究者讨论任何具体条目。我们需要知道不同的人独立阅读同一条发言时，判断是否一致，因此确认可能会让这份工作失去意义。

## 2. 标签体系

标签分为三组，共十一个栏目。

### 第一组：购买理由（2 个）

回答的问题是：**这条发言在用什么理由支持或否定购买某个品牌？**

**① 价值认同型**<br>
理由指向产品本身：成分、安全性、功效、使用感受、性价比、科研背书。<br>
判断技巧：把品牌名遮住，这句话是否仍然在谈“这个东西对我的皮肤或头发做了什么”。

|    | 发言                    | 说明                                    |
| -- | --------------------- | ------------------------------------- |
| 算  | “这款成分温和，我烂脸期全靠它”      | 理由指向使用效果                              |
| 算  | “这玩意有矿油，很毁皮肤”         | 批评产品也算，因为理由仍指向产品本身                    |
| 算  | “人家是真的有实验室有专利的”       | 理由指向科研背书                              |
| 不算 | “抵制日货！不去日本！”          | 只有立场，完全没谈产品好坏                         |
| 不算 | “日本品牌避雷清单：CPB、怡丽丝尔……” | 只是列名单示警，没有对产品属性做评价（根据上下文，单纯示警归安全恐慌，带立场攻击归立场抵制） |
| 不算 | 只是提到某个品牌名或出现在话题标签里    | 提到品牌 ≠ 用产品属性做购买理由                     |

<br>

**② 社会信号型**<br>
理由指向产品的社会含义：用这个牌子在别人眼里代表什么、有没有面子、别人是否都在用。<br>
判断技巧：把产品好不好用遮住，这句话是否仍然在谈“用这个东西的我在别人眼里是什么样”。

|    | 发言                     | 说明                                           |
| -- | ---------------------- | -------------------------------------------- |
| 算  | “贵妇标配，拿得出手”            | 理由指向身份象征                                     |
| 算  | “其实不算好用，但毕竟是大牌，拿出去有面子” | 明确以面子为由                                      |
| 算  | “别为了充面子跟风买贵妇面霜”        | 反向提到面子动机，也算                                  |
| 不算 | “这个牌子很有名很多人用”          | 只是陈述知名度，没有指向“用它代表我是什么人”。除非发言把“别人都用”当作自己购买的理由，例如“这么多人用，皮肤科医生都推荐，说明它确实好用”理由指向产品功效被社会认可，属于价值认同型。 |
| 不算 | 单纯晒出高价产品、没有任何身份含义的表达   | 晒物 ≠ 以社会信号为购买理由。一张贵妇面霜的照片配一句“今天到货”，很难判断是是炫耀还是记录，是开箱习惯还是给闺蜜看的，读者之间没有共识
> 说明：社会信号型在这批语料里本来就少见，遇到拿不准的，可以回想判准中“是不是在谈‘别人怎么看我’”。如果谈的是产品效果就不是这一类。

<br>

### 第二组：对事件的反应（7 个）

回答的问题是：**这条发言对“日本核污染水排海”这件事表现出了什么反应？**<br>
如果发言完全没有涉及这个事件（例如单纯的产品测评、日常种草），这一整组都留空。

**③ 安全恐慌**：担心产品因核污染而危害健康。关注点在**产品是否安全**上。

|    | 发言                 | 说明                           |
| -- | ------------------ | ---------------------------- |
| 算  | “这个还能用吗，不敢用了”      | 担心产品安全                       |
| 算  | “查了一下批次是不是排海之前生产的” | 用行动规避安全风险                    |
| 不算 | “从此资生堂退出我的梳妆台”     | 这是决定放弃，没有表达对安全的担心，更接近立场或转投 |
| 不算 | “绝不补货”             | 表达的是态度或抵制，不是对产品安全的担忧         |

<br>

**④ 立场抵制**：因为对日本这个国家的立场、愤怒而拒绝购买，**与产品是否真的受污染无关**。关注点在**国家和立场**上。

这是上一轮分歧最大的标签，请特别注意三条边界：

* 必须有明确的“因为是日本 / 因为国家立场”的指向。**只是提到核污水、或只是这条发言跟事件有关，都不算立场抵制。**
* 纯新闻转述（只陈述事实、没有发言者自己的态度）不标任何态度标签；如果整条只是转述新闻、与购买讨论无关，归“主题无关”。
* “舍不得但不敢用了”这种矛盾心情，属于割舍挣扎，不是立场抵制。

|    | 发言                        | 说明                |
| -- | ------------------------- | ----------------- |
| 算  | “不买不是因为辐射，是因为日本这两个字”      | 明确指向国家立场          |
| 算  | “还买日货的都是什么人”              | 用立场审判他人消费         |
| 算  | “辐人居辐地，辐地辐人居”             | 谐音玩梗，但立场明确（诅咒式抵制） |
| 不算 | “资生堂股价大跌，称核污水排海影响中国需求……”  | 纯新闻转述，无个人态度       |
| 不算 | “因为核辐射吗，我还想买悦薇水乳呢”        | 是舍不得（割舍挣扎），不是抵制   |

<br>

**⑤ 理性辩护**：用讲道理的方式论证产品仍然安全，或认为大家反应过度。

|    | 发言                | 说明                          |
| -- | ----------------- | --------------------------- |
| 算  | “进口商品都要经过海关检测”    | 用监管论证安全                     |
| 算  | “吃进去的都不怕，涂皮肤上怕什么” | 用剂量常识反驳恐慌                   |
| 算  | “别制造焦虑了”          | 认为反应过度                      |
| 不算 | “我觉得没事继续用”        | 只有结论没有论证，不算辩护（可能属于摆烂虚无，看语气） |

<br>

**⑥ 摆烂虚无**：用“反正躲不掉”“无所谓了”的态度回应，既不辩护也不抵制，放弃认真对待。

|    | 发言                  | 说明               |
| -- | ------------------- | ---------------- |
| 算  | “污染三个月就扩散全世界了，爱咋咋吧” | 放弃抵抗             |
| 算  | “已经摆烂了，死不了”         | 虚无态度             |
| 不算 | “海关会检测的没事”          | 这是讲道理（理性辩护），不是摆烂 |

<br>

**⑦ 割舍挣扎**：明确表达“想换掉但换不掉”“舍不得”的矛盾心情。

|    | 发言                       | 说明                  |
| -- | ------------------------ | ------------------- |
| 算  | “说好不用日货，但换了别的真的会晒黑，纠结死了” | 矛盾心情                |
| 算  | “找了很久替代品，最后还是回购了”        | 想放弃但没做到             |
| 算  | “因为核辐射吗，我还想买悦薇水乳呢”       | 舍不得，典型的挣扎           |
| 不算 | “已经换成国货了，很满意”            | 已经干脆换掉，没有挣扎（属于替代转投） |

<br>

**⑧ 替代转投**：因为**这次事件**而换用其他品牌，或因事件而询问、寻找替代品。**关键是要和这次事件有关。**

|    | 发言               | 说明              |
| -- | ---------------- | --------------- |
| 算  | “因为核废水，已经换成国货了”  | 因事件而转投          |
| 算  | “日货不敢买了，有没有替代推荐” | 因事件而找替代         |
| 不算 | “敷尔佳还不错，种草了”     | 普通种草，和事件无关      |
| 不算 | “跟着某某博主买准没错”     | 普通推荐，和事件无关      |
| 不算 | 任何脱离核污水事件的日常安利   | 种草 ≠ 转投，必须由事件驱动 |

<br>

**⑨ 国货红利**：借这件事表达对中国品牌的支持、赞美或身份认同。

|    | 发言               | 说明                       |
| -- | ---------------- | ------------------------ |
| 算  | “国货之光，支持国产”      | 赞美国货                     |
| 算  | “国货不嫌我穷，我凭什么嫌弃它” | 身份认同                     |
| 不算 | “国货真不给力”         | 批评国货，不是红利（按情况归价值认同型负面评价） |

<br>

### 第三组：无法归类的情况（2 个）

有些内容经过脚本清洗后仍然残留，可能不含任何与本项目相关的信息。这两栏是给这类内容准备的正当出口。**勾选这两栏可以帮助我们更好地识别数据在机械清洗之后仍然留存的噪音。**如果这两类不足以涵盖所有的无法判断，欢迎继续通过“无法判断”与填写备注向我们反馈。

**⑩ 主题无关**：发言与美妆购买、核污水事件都无关。例如追星、日常生活闲聊、纯广告推销、与本项目无关的吐槽。

|   | 发言           | 说明         |
| - | ------------ | ---------- |
| 算 | “今晚八点直播不要错过” | 主播口播，与主题无关 |
| 算 | 追星、饭圈、无关生活琐事 | 与美妆和事件都无关  |

<br>

**⑪ 上下文缺失**：发言本身可能有意义，但脱离图片、视频、链接就无法判断。

|   | 发言       | 说明        |
| - | -------- | --------- |
| 算 | “图一支持了”  | 看不到图，无法判断 |
| 算 | “最后一张是？” | 依赖看不到的图片  |

> 主题无关和上下文缺失的区别：前者是“内容和项目无关”，后者是“内容可能相关，但缺了看不到的东西所以没法判断”。拿不准两者选哪个时，两个都不勾也可以，在备注里写一句原因。

## 3. 标签之间怎么切：几组最容易混的对照

当一条发言同时像两个标签时，用下面的规则切分：

* **安全恐慌 vs 立场抵制**
  * 担心的是“产品脏了不安全”→安全恐慌
  * 针对的是“日本这个国家”→立场抵制
  * 两种味道都明确有，就两个都标
* **价值认同型 vs 只是提到品牌**
  * 对产品的成分、效果、安全做了评价→价值认同型
  * 只是出现了品牌名或话题标签→不属于该类别
* **替代转投 vs 普通种草**
  * 由核污水事件驱动的换牌、找替代→替代转投
  * 与核污水事件无关的日常安利→不标（第二组整组留空）
* **割舍挣扎 vs 替代转投**
  * 想换但没换成、还在纠结→割舍挣扎
  * 已经干脆换掉→替代转投
* **理性辩护 vs 摆烂虚无**
  * 讲了道理（检测、剂量、监管）→理性辩护
  * 只是“无所谓、躲不掉”→摆烂虚无

其他提醒：

* **反讽和阴阳怪气**：按发言的实际立场标，不按字面。“干得漂亮”如果是在为日系品牌销量暴跌叫好，属于立场抵制。
* **转发内容带简短评论**：只标注发言者自己写的那部分表达的态度。
* **多标签是正常的**：一条发言“才跌 16%？那些人不怕得癌吗”，同时有安全恐慌（怕得癌）和立场抵制（对销量没跌到底的愤怒），两个都标。不要强行二选一。

## 5. 操作说明

1. 表格共有若干行，每行一条发言。请按顺序标注，不要跳行。
2. 每个标签列都是一个下拉单元格：点击单元格右侧的下拉箭头，选择 **✓** 表示该标签适用；**不适用的列保持空白**（不要手动输入其他符号）。一条发言适用几个标签就在几列选 ✓。
3. 备注列自由填写：犹豫的原因、看不懂的词、觉得定义有问题的地方，都欢迎写。
4. 请分多次完成，连续标注建议不超过 30 分钟，疲劳会显著影响判断质量。
5. 完成后将文件发回给项目负责人。请不要修改表格结构（不要增删行列、不要排序）。

## 6. 关于这一轮标注

本轮请你在阅读上面的新定义后，对收到的样本**从头独立标注一遍**。即使其中有些发言你上次可能见过，也请按现在的定义重新判断，不必回想上次怎么标的。这一轮的目的是检验清晰的定义能否让不同标注者的判断更接近，因此你此刻独立、按新定义做出的判断，正是我们需要的。
